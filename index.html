NirvanaPass

Password management for the 21st century

NirvanaPass is a new kind of password manager designed to eliminate storing and remembering passwords. More details will be made available in due course of time.

NirvanaPass is coming to a device near you in the first quarter of 2016 sometime in the near future. I hope to make available apps for Android (native Java), Windows (Universal Windows Platform), iOS (native Swift) and the command line (either Java or Python). While all four might not be available at once, I think an Android app and a command line port will be the first to arrive.

Follow the updates on a blog I have setup at https://blog.nirvanapass.com. For the social media fanatics, there is Facebook and Twitter.

See you soon!
